The new code.

Zip file containing Dunit2 full source including 3rd party code FastMM and JEDI_JCL



Notes.

 DUnit2 has not yet been subjected to scruitiny by members of the dunit-interest group and thus has no official status.
The group and list coordinator are kept informed of developments but have not begun a public examination or discussion on the merits of the new code.
It's unofficial name DUnit2 is also open to change if necessary.
All code derived from prior work is acknowledged in the source files.
