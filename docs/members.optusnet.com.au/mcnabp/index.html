Advanced Dunit development.

Codegear have release Rad Studio 2009 with Generics and WideString Unicode support.
DUnit2 has been updated and several unit tests adjusted to compile under D2009 (Win32).
All tests pass.

A separate project folder has been created for D2009 Win32.
The latest Jedi JCL sourcecode appears to be work in progress and is not yet compiling under D2009.
Consequentially DUnit2's D2009 project options currently exclude the USE_JEDI_JCL compiler option.
